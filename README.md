# Convenience-store-management-system

POS system: Backend API

### Feature 
> - Create order
> - Calculate discount 
> - Add membership
> - Add product
> - Add promotion
> - Check how many products sold


### Tech Stack 
> -  Use Golang as Backend  
> -  Use kafka as message queue
> -  Use mySQL as Database
